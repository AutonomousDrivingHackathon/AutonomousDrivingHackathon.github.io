Simulation environment and available resources:

We will be using a highway driving simulator which can be found at: https://github.com/eleurent/highway-env . The simulator is designed for reinforcement learning.

However, for this competition, you can use any machine learning techniques or simply write "if--else--" rules, or a combination of any approaches.

We will evaluate submitted code in an "agent-environment interactions" manner. More details will be announced regarding the submision format and evaluation metrics.

We will provide simple tutorials to help you get started. This includes the following:

1. How to setup the simulator environment.

2. Write a basic deep Q-learning (reinforcement learning) agent.

3. Basic ideas to make modifications of a standard deep Q-learning agent. This way, you can customize your own agent for competition.

4. More on demand.
